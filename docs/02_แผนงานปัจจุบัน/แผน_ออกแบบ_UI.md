## 📱 หน้าทั้งหมด

| หน้า | Path | คำอธิบาย |
|---|---|---|
| **หน้าหลัก** | `/` หรือ `index.html` | Avatar 3D + ข้อความต้อนรับ |
| **สถานที่** | `/places.html` | รายการสถานที่ท่องเที่ยว |
| **แชท** | `/chat.html` | Chat กับน้องน่าน |
| **เกี่ยวกับ** | `/about.html` | เกี่ยวกับโปรเจกต์ |
| **Admin** | `/admin/` | สำหรับ Admin (แยก folder) |

## 🎨 Design System

### สี
| ตัวแปร | ค่า | ใช้สำหรับ |
|---|---|---|
| `--primary` | #40c4ff | ปุ่มหลัก, ไฮไลท์ |
| `--secondary` | #ec4899 | Accent |
| `--bg` | #0f0c29 | พื้นหลัง |
| `--text` | #ffffff | ข้อความ |

### Typography
- **Font:** Kanit
- **Headings:** 600 weight
- **Body:** 400 weight
