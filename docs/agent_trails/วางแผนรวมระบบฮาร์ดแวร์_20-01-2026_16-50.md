# บันทึกความคืบหน้า: วางแผนรวมระบบฮาร์ดแวร์

**วัตถุประสงค์ (Task Objective):**
- วางแผนการนำโค้ด Arduino ที่ใช้ micro-ROS มาใช้งานร่วมกับระบบหุ่นยนต์
- ออกแบบตัวเชื่อมต่อ (Kinematics Bridge) เพื่อควบคุมมอเตอร์ทั้ง 4 ล้อ

**การดำเนินการ (Action Taken):**
- ตรวจสอบโครงสร้างโปรเจกต์เดิม
- วิเคราะห์โค้ด Arduino (ESP32) ที่ให้มา:
    - ใช้ micro-ROS สื่อสารแบบ Publisher/Subscriber
    - มีระบบ PID Control สำหรับ 4 มอเตอร์
    - ใช้ขาพิน (Pins) ต่างๆ บน ESP32 สำหรับ Motor Driver และ Encoder
- ตรวจสอบความพร้อมของ ROS 2 ในระบบ (ติดตั้งแล้ว)

**ผลลัพธ์ (Outcome):**
- ได้แนวทางการสร้าง `kinematics_bridge.py` เพื่อรับค่าจาก `/cmd_vel` มากระจายให้มอเตอร์แต่ละตัว
- กำหนดโครงสร้างโฟลเดอร์สำหรับ Hardware เก็บไว้ที่ `hardware/`

**สถานะปัจจุบัน (Current State):**
- กำลังจัดทำ `implementation_plan.md` เพื่อส่งให้ผู้ใช้รีวิว

**ขั้นตอนต่อไป (Next Step):**
- สร้างไฟล์ `implementation_plan.md` พร้อมรายละเอียดการคำนวณ Kinematics และโครงสร้างไฟล์
